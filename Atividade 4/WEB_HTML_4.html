Exercicio 4

O HTML é uma linguagem de marcação usada para estruturar e apresentar o conteudo na web.

Ele foi criado em 1991 por Tim Bernes-Lee, e desde então tem evoluido constantemente. Atualmente, é fundamental para a criação de sites e aplicativos modernos

O uso de tags é essencial para formatar o conteudo textual corretamente.
